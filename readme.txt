v16.13.0 Versiyonlu NodeJS kurulu olmalıdır.
NodeJS kurulduğundan emin olduktan sonra bu dizine geçip `npm install` komutu ile Node paketlerini kurun.
Paketlerin kurulduğundan emin olmak istiyorsanız, dizinde `node_modules` adlı klasör olup olmadığını kontrol ediniz.
Her şeyin doğru olduğundan emin olduktan sonra ise `npm start` yazarak uygulamayı çalıştırınız.
